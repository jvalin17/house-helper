# Resume Templates

Place your reference resume here. The app will use it as the default format.

## How to add a template

1. Drop your best resume as `default.docx` or `default.pdf` in this folder
2. The resume builder will follow its section order, formatting style, and layout
3. You can add multiple templates:
   - `junior.docx` — skills-first format
   - `mid.docx` — balanced format
   - `senior.docx` — impact-first format

## Supported formats

- `.docx` — parsed for section structure and formatting
- `.pdf` — parsed for text layout
- `.md` — used directly as markdown template

## What the app learns from your template

- Section ordering (which sections come first)
- Bullet point style (how achievements are written)
- Length (how many bullets per role)
- What to include vs omit
